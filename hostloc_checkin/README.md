### 作用
在hostloc.com（全球主机交流论坛）的页面上增加签到按钮（按钮在网页右上个人信息附近，见图片）。
点击签到按钮，通过js模拟访问其他用户空间，增加金钱数量。
点击签到按钮后，按钮上会有进度显示，当显示为100%时为已访问10个页面，获得全部靠访问空间可以获的积分。
